Bienvenue sur la Page de Test de l'App

Pour tester l'application, suivez les instructions ci-dessous :

Connexion avec Google

Vous pouvez vous connecter à l'application en utilisant votre compte Google. Cette connexion est essentielle pour tester toutes les fonctionnalités de l'app.

Scannez les QR Codes

Scannez l'un des QR codes ci-dessous pour accéder à un produit. Si vous souhaitez ajouter un deuxième animal par la suite, scannez le deuxième QR code.
Les codes QR sont valables 7 jours à compter du 24/04/2025.

Test des Abonnements via Stripe

Nous avons mis Stripe en mode test pour que vous puissiez tester les paiements d'abonnements. Utilisez les informations suivantes pour effectuer un paiement de test :

Voici quelques autres cas de paiement à tester :

    Numéro de carte : 4242 4242 4242 4242

    Numéro de carte : 4242 4242 4242 4242
